Cara upload ke GitHub Pages:
1. Buka github.com lalu login.
2. Buat repository baru, contoh nama: untuk-tya.
3. Upload semua file/folder ini ke repository: index.html, style.css, script.js, folder assets.
4. Masuk Settings > Pages.
5. Pada Source pilih Deploy from a branch.
6. Branch pilih main dan folder /root, lalu Save.
7. Tunggu sebentar, link akan muncul dengan format:
   https://USERNAME.github.io/untuk-tya/

Ganti foto:
- Masukkan foto ke folder assets.
- Namai: foto1.jpg, foto2.jpg, foto3.jpg, foto4.jpg, foto5.jpg, foto6.jpg

Ganti lagu:
- Masukkan lagu ke folder assets.
- Namai: lagu.mp3

Catatan:
Browser biasanya tidak mengizinkan musik autoplay, jadi musik akan mulai setelah tombol Mulai Cerita Kita ditekan.
